PES Project 1 README
---------------------------------

Team Members: Mike Fruge and Mark Hinkle

Repo Description:
	This repository includes any source or include files used in creating solutions to PES
Project 1 and the captured output from each program.

Instillation Instructions:
	Source files should be compiled with the command "gcc <SourceFileName> -Wall -Werror"
This compilation will produce an executable named a.out, which when run will produce the results recorded in the PDF's.
